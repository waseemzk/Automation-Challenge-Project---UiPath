# Automation Challenge - UiPath
## 📖 Overview
This project is an automation solution built using UiPath to solve the Automation Challenge.
The bot reads structured data from an Excel file and automatically fills a dynamic web form with high accuracy and speed.
<br><br>
<img src="./giphy.gif" width="300"/>

## 🎯 Objectives
- Automate repetitive data entry tasks <br>
- Handle dynamic selectors in web applications <br>
- Improve efficiency and reduce human error <br>

## ⚙️ Features
- Read data from Excel file
- Automate web form submission
- Handle dynamic UI elements

## How to Run
- Install UiPath Studio
- Clone this repository
- Open the project in UiPath
- Make sure the Excel file path is correct
